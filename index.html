SIMON COLOR GAME


INTRODUCTION

The player must carefully watch the sequence and repeat it by clicking the buttons in the exact same order. With each new level, the game adds one more random color to the sequence, making it progressively more challenging. The game continues until the player clicks the wrong button.


RULES

  1. Press start button.
  2. One button flashes to indicate the first color in the sequence.
  3. Click the flashed button correctly.
  4. If your input is correct, the game advances to the next level.
  5. At each new level, the game adds one new random color to the existing sequence.
  6. Repeat the entire sequence in the correct order.
