# 🎮 Simon Color Game

> **"Trust your memory... until it betrays you." 😈**

Welcome to the **Simon Color Game**, where four innocent-looking buttons are secretly plotting against your memory.

Every level adds one more color to the sequence.
Your job?
Remember it.
Repeat it.
Don't panic.

(You *will* panic.)

---

## 🎯 How to Play

1. Press **any key** to start.
2. Watch which color flashes.
3. Repeat the sequence by clicking the buttons.
4. Each level adds a new color.
5. Miss one click...
6. 💀 **Game Over.**

Simple?
Go ahead... prove it.

---

## ✨ Features

- 🎨 Four colorful interactive buttons
- ⚡ Smooth flash animations
- 📈 Unlimited levels
- 🏆 High Score Tracker
- 💥 Red flash on Game Over
- 🔁 Instant restart
